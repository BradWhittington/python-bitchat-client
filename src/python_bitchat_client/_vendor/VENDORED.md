# Vendored dependency: bitchat-protocol

This directory contains a vendored copy of the upstream `bitchat-protocol` Python package.

- Upstream package: `bitchat-protocol`
- Upstream version: `0.1.1`
- Upstream source: `https://github.com/bitchat-sdk/protocol-core-py`
- Imported from: PyPI source distribution (`bitchat_protocol-0.1.1.tar.gz`)

Vendored files are intentionally isolated under `python_bitchat_client._vendor`.
When updating upstream code, prefer replacing files from a tagged upstream release
instead of rewriting behavior locally.

The upstream project license and README are stored as:

- `src/python_bitchat_client/_vendor/bitchat_protocol_LICENSE`
- `src/python_bitchat_client/_vendor/bitchat_protocol_README.md`
